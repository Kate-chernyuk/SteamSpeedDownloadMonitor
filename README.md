Скрипт на Python для замера скорости загрузки игр на Steam. 
Работает независимо от места установки Steam (путь достаётся из реестра Windows). 
Информация собирается из файла логов Steam: logs/content_log.txt
Выводит в консоль текущую скорость, среднюю скорость, статус, название и id игры.
Название игры извлекается с помощью ID из appmanifest_{app_id}.acf.
Данные обновляются каждую минуту в течение 5 минут.

# Требования
1) Windows
2) Установленный Steam
3) Python 3.9+ (подойдёт и новее)
